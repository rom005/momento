# Momento — AI Travel Memory Platform

Momento is a an AI Travel Memory Platform designed to showcase:

- **Rust backend engineering**
- **Distributed systems**
- **AI systems engineering**
- **Observability**
- **Reliability**
- **Product engineering**

## What Momento Does

Momento helps travelers capture and reuse meaningful experiences by storing:

- Memories
- Notes
- Locations
- Photos
- Trips

## AI-Powered Features

Momento turns raw travel data into useful content:

- Generate blogs from saved trip memories
- Generate trip summaries
- Generate social posts
- Semantic search across all stored travel context
- Conversational retrieval for natural-language Q&A over your travel history

## Product Goals

- Fast, reliable memory capture
- Rich retrieval and discovery
- High-quality generated travel narratives
- Production readiness through strong observability and resilient architecture

## Vision

A durable travel memory system where users can save everything once, then rediscover, ask, and publish instantly using AI.
